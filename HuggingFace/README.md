# 🥔 Potato Disease Classification

Aplikasi untuk mendeteksi penyakit pada daun kentang menggunakan CNN.

## Kelas:
- Early Blight
- Late Blight  
- Healthy

## Cara Pakai
1. Upload gambar daun kentang
2. Klik tombol "Prediksi"
3. Lihat hasil klasifikasi
